ChatGPT 4o with Canvas Executor

Steps:

- `brew install npm`
- `node server/app.js`
- Go to `chrome://extensions/`
- Load unpacked
- Point to CanvasExec/chrome_ext
- Open any chat with canvas and enjoy!


To support running various languages, run these as needed
- `brew install python`
- `brew install gcc`
- `brew install npm`
